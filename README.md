# Spring Boot Hello REST API

## Overview
This is a simple Spring Boot REST API built using Java 21.
It demonstrates basic REST concepts and controller mappings.

## Tech Stack
- Java 21
- Spring Boot
- Maven
- REST API
- Git & GitHub

## Endpoints

### GET /hello
Returns a welcome message.

### GET /hello/{name}
Returns a personalized welcome message.

## How to Run
```bash
mvn spring-boot:run
